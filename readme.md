## Setup

1. Clone the repository:

```bash
git clone https://github.com/shaurya35/Stockwise-Inventory-Manager
cd Stockwise-Inventory-Manager
```

2. Install the dependencies:

```bash
npm install
```

## Usage

```bash
npm run dev
```

Access the app in your web browser at `http://localhost:5173/`.
